Provide the points/vertices in the vertex.txt file
The file format should be like this : 
The first line should indicate the number of points in the polygon
The next subsequent lines should be the co-ordinates of the point seperated by ,
And the vertices must be given either in clock-wise or anti-clock-wise direction


Example : 
For three points the file should be like this
3
8,0
0,4
12,4
